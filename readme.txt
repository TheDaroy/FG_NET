How to build:
1. Right click on FGNet.uproject, select 'Generate Visual Studio project files'
2. Start FGNet.sln
3. Make sure FGNet is selected as Startup Project, to verify this, the text will be bold. If not, right click on FGNet and select 'Set as Startup Project'
4. Select DebugGame Editor in the Solution Configuration, and Win64 as Platform.
